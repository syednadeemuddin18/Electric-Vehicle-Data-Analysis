# Electric Vehicle Data Analysis

## Overview
This project analyzes electric vehicle registration data to understand adoption trends, regional distribution, and vehicle characteristics.

## Dataset
Washington State Department of Licensing – Electric Vehicle Population Data

## Objectives
- Clean and preprocess EV data
- Analyze EV adoption trends
- Study regional and county-level distribution
- Visualize insights using charts and maps
- Discuss linear regression for electric range prediction

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Files in this Repository
- `EV_Data_Analysis.ipynb` – Data analysis and visualizations
- `Electric Vehicle Data_Analysis.pdf` – Final report

## Key Insights
- Tesla dominates EV adoption
- Urban counties show higher EV concentration
- EV adoption increased sharply after 2020
- Average electric range is ~109 miles
